# 项目规定
> Talk is cheap, show me the code.
Github仓库地址：https://github.com/HangYaHan/FeedbackTrader

项目约定：
1. 为保持main分支干净，所有的功能开发和debug必须新建分支完成。
2. src文件夹中不应该包含一个中文字符
3. 对于一个PR，应该提交一份包含主要函数摘要的md文件，至少包括参数，返回值和函数功能的描述
4. 对于bug和todo，在GitHub Issues中发布。任何想法都可以在上面发布，建议立刻发布以免之后忘了。
